## ğŸš€ Quick Start

### Install globally

### Initialize your project

### Add components
```bash
# Add a single component
ngsui-cli add button

# Add multiple components  
ngsui-cli add button badge alert

# List all available components
ngsui-cli list

# Add all available components
ngsui-cli add --all

```
